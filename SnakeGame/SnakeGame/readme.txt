Snake Game

Overview
The Snack Game is a simple web-based game where players control a snake to collect snacks and grow longer. It is built using HTML, CSS, and JavaScript.

Features
Snake Movement: Use arrow keys to control the snake's movement.
Snack Collection: The snake grows longer when it collects snacks.
Collision Detection: The game ends if the snake collides with the boundaries or itself.
How to Play
Clone the Repository:

git clone https://github.com/your-username/snack-game.git
Open index.html in a Web Browser:

Locate the project folder on your computer.
Open the index.html file with your preferred web browser.
Start Playing:

Use the arrow keys (up, down, left, right) to control the snake.
Collect snacks to grow longer.
Avoid collisions with the boundaries and the snake itself.
